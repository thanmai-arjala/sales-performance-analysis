# Sales & Profit Performance Analysis

## Overview
This project analyzes retail sales data from the Superstore dataset to understand sales performance, profitability, and customer segments. The objective is to derive actionable business insights using data analysis techniques.

## Dataset
- Superstore Sales Dataset
- Contains sales, profit, category, region, and customer segment information

## Tools & Technologies
- Python (pandas)
- Jupyter Notebook
- GitHub

## Key Analysis Performed
- Total sales and profit analysis
- Sales performance by product category
- Profit analysis by region
- Sales contribution by customer segment

## Key Insights
- Technology is the highest revenue-generating category.
- The West region contributes the highest overall profit.
- The Consumer segment accounts for the largest share of total sales.

## Files in this Repository
- `sales_analysis.ipynb` – Data cleaning, analysis, and insights
- `SampleSuperstore.csv` – Dataset used for analysis

## Conclusion
This analysis highlights key drivers of sales and profitability and demonstrates practical data analysis skills relevant to entry-level Data Analyst roles.
